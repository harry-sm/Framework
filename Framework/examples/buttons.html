Buttons

Defualt Buttons

To render a default button add class .btn to any html element you wish to be viewed as a button 

                        <button class="btn flat sk-grn has-states">Button</button>
                        <button class="btn flat sk-blu has-states">Button</button>
                        <button class="btn flat sk-org has-states">Button</button>
                        <button class="btn flat sk-red has-states">Button</button>
                        <button class="btn flat sk-gry has-states">Button</button>

Stacked Buttons

Add class btn-group-statcked to wrapper element and place buttons inside.

                        <div class="btn-group-statcked">
                            <button class="btn flat sk-grn has-states">Button</button>
                            <button class="btn flat sk-blu has-states">Button</button>
                            <button class="btn flat sk-org has-states">Button</button>
                            <button class="btn flat sk-red has-states">Button</button>
                            <button class="btn flat sk-gry has-states">Button</button>
                        </div>

Inline Buttons

Add class btn-group-inline to wrapper element and place buttons inside.

                        <div class="btn-group-inline">
                            <button class="btn flat sk-grn has-states">Button</button>
                            <button class="btn flat sk-blu has-states">Button</button>
                            <button class="btn flat sk-org has-states">Button</button>
                            <button class="btn flat sk-red has-states">Button</button>
                            <button class="btn flat sk-gry has-states">Button</button>
                        </div>

Fluid Inline Buttons

Add classes btn-group-inline and fluid to wrapper element. Place each button element into a fluid-btn element. See example below

                        <div class="btn-group-inline fluid">
                            <div class="fluid-btn">
                                <button class="btn flat sk-grn has-states">Button</button>
                            </div>
                            <div class="fluid-btn">
                                <button class="btn flat sk-blu has-states">Button</button>
                            </div>
                            <div class="fluid-btn">
                                <button class="btn flat sk-org has-states">Button</button>
                            </div>
                            <div class="fluid-btn">
                                <button class="btn flat sk-red has-states">Button</button>
                            </div>
                            <div class="fluid-btn">
                                <button class="btn flat sk-gry has-states">Button</button>
                            </div>
                        </div>

Size Buttons

Add class sz- to button. sz-xs - extra small sz-sm - small sz-lg - large sz-xl - Extra large

                        <button class="btn flat sz-xs sk-grn has-states">Button</button>
                        <button class="btn flat sz-sm sk-blu has-states">Button</button>
                        <button class="btn flat sk-org has-states">Button</button>
                        <button class="btn flat sz-lg sk-red has-states">Button</button>
                        <button class="btn flat sz-xl sk-gry has-states">Button</button>

Disabled Buttons

Add the disabled attribute or class .disabled to button to prevent user interaction.

Button
Button
                        <button class="btn flat sk-grn has-states" disabled>Button</button>
                        <a href="#" class="btn flat sk-blu has-states disabled">Button</a>
                        <div class="btn flat sk-org has-states disabled">Button</div>
