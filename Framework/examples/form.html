Fields

To add styling to forms element wrap form element in elements with .field-block class and add class .field-item. This does not apply to raido and checkboxes

Field

This is a default input field.

                        <div class="field-block">
                            <label for="">label</label>
                            <input type="text" class="field-item">
                        </div>

Textarea

This is a default textarea.

                        <div class="field-block">
                            <textarea class="field-item" rows="10"></textarea>
                        </div>

Select Field

This is a default select field.

                       <select class="field-item">
                           <option value="mango">Mango</option>
                           <option value="carrot">Carrot</option>
                       </select>

Mutiple Select Field

This is a default Mutiple Select Field.

                       <select class="field-item">
                           <option value="mango">Mango</option>
                           <option value="carrot">Carrot</option>
                       </select>

Checkboxs and radios

To add styling to checkboxs and radios wrap form element in elements with .field-block and then .radio or .checkbox. See example for reference

Checkbox

This is a default checkbox.

                       <div class="field-block">
                          <label for="cold">	
                              <div class="check">
                                  <input id="cold" name="cold" type="checkbox" value="option1">
                                  <span></span>
                              </div>
                              Cold
                          </label>
                       </div>

Radio

This is a default radio.

                       <div class="field-block">
                          <label for="blue">	
                              <div class="radio">
                                  <input name="radio" id="blue" type="radio" value="option2">
                                  <span></span>
                              </div>
                              blue
                          </label>
                       </div>

Fields With Addons

Field

Place element with .field-addon and sibbling of .field-item inside a .field-block element.

Search 
                        <form>
                            <div class="field-block">
                                <input type="text" class="field-item">
                                <span class="field-addon">Search</span>
                            </div>
                        </form>

Fields States

Disabled

Add the disabled attribute to input to prevent user input.

                            <div class="field-block">
                                <label for="">label</label>
                                <input type="text" class="field-item" disabled>
                            </div>

Read Only

Add the readonly attribute on an input to prevent user input .

                            <div class="field-block">
                                <label for="">label</label>
                                <input type="text" class="field-item" readonly>
                            </div>

Required

Add the required attribute on an input to indicate to user input is required.

                            <div class="field-block">
                                <label for="">label</label>
                                <input type="text" class="field-item" required>
                            </div>

Aligned

Add class field-group-align to container inside the form element and place field element inside it to triger layout.

                            <form>
                                <div class="field-group-align">
                                   < ... />
                                </div>
                            </form>
