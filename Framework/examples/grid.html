Grid

Configuring the LESS variables

The grid defaults to a 24 column (bricks), 960 pixel grid system. Customize the columns, gutters, grid with unit type(px, em, %, rem). Variables are loacted in grid.less file

                    @wall: 960;
                    @bricks: 24;
                    @gutter: 10;
                    @unit: 1px;

Auto Generate Grid

You can auto generate all the classes for the grid by importing the gird.less file to your working file and calling the autoGrid mixin in your less file. You can also pass in a parameter that makes you name the class of your columns

                    @import 'grid.less';
                    
                    .autoGrid(column);

Semantic Use

By using the bricks() mixin You can take advange of sematics. The mixin comes with four parameters: @brick the column number you want to render, @frame change the context of the grid width to the column with which allows for nesting, @gutter the space between the columns, @unit type of unit of measure 

                    .class{
                        .brick( @brick, @frame:@bricks, @gutter:@gutter, @unit:@unit );
                        ...
                    }
Parameters Description
@brick The column number you want to render
@frame Change the context of the grid width to the column with which allows for nesting
@gutter The space between the columns
@unit Type of unit of measure (px, em, rem, %)

Less

                    .left-col {
                        .brick(5)
                    }
                    .main {
                       .brick(14)
                    }
                    .right-col {
                       .brick(5)
                    }

Push and Pull

.push() and .pull()Use to source order columns. Pass in the column number in the parameter you wish to move that column to that number of column spaces.

Left

Center

Right

                    .left-col {
                        .brick(5);
                        .push(19);
                    }
                    .main {
                       .brick(14);
                       .pull(5);
                    }
                    .right-col {
                       .brick(5);
                       .pull(5);
                    }
