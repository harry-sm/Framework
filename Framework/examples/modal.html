Modals

The plugin exposes three methods designed to mimic their native JavaScript equivalents. They take various parameters to customise specify defaults. Most common usage is $(element).modal( type, options, callback ). available types includes alert, confirm, prompt.

                              {

                              },
                              function (result){
                                  if(result)
                                      modalDemo.show("continue")
                                  else
                                      modalDemo.show("stop");
                              });

Options

Options Default Context Description
debug true - Outputs error meaages or warnings to the console
close '.modal-close' All Close button selector
animate scaleIn All Animation type
body '.modal-body' All Modal body selector
message empty All Create custome message in modal
ok '.ok' All OK button selector
cancel '.cancel' Confirm & Prompt Cancel button selector
input '.prompt-input' Prompt Input Box selector

Callbacks

Callbacks specify a function to occur after a specific behavior.

Type Description
alert Basic callback. Does not pass any data
confirm Callback passes data of result true or false
prompt Callback passes data of result input box

Methods

Methods Description
Destroy Removes all changes to the page made by initialization
reInitilize Refreshes cached values for a component after Destroy is called
