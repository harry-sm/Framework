Table

All table have a width of 100% tto make the fluid

Table Default

To style table, add .table class. This class adds padding and border bottom to table elements.

# Name Age Gender
1 Warren Buffet 90 Male
2 Steve Jobs 50 Male
2 Narine Smith 30 Female 
                        <table class="table">
                            <colgroup>
                                <col width="40">
                                <col>
                                <col>
                                <col>
                            </colgroup>
                            <thead>
                                <tr>
                                    <th>#</th>
                                    <th>Name</th>
                                    <th>Age</th>
                                    <th>Gender</th>
                                </tr>
                            </thead>
                            <tbody>
                                <tr>
                                    <td>1</td>
                                    <td>Warren Buffet</td>
                                    <td>90</td>
                                    <td>Male</td>
                                </tr>
                                <tr>
                                    <td>2</td>
                                    <td>Steve Jobs</td>
                                    <td>50</td>
                                    <td>Male</td>
                                </tr>
                                <tr>
                                    <td>2</td>
                                    <td>Narine Smith</td>
                                    <td>30</td>
                                    <td>Female</td>
                                </tr>
                            </tbody>
                        </table>

Table Bordered

To add borders to all cells, add .is-bordered class to the table.

# Name Age Gender
1 Warren Buffet 90 Male
2 Steve Jobs 50 Male
2 Narine Smith 30 Female 
                        <table class="table is-bordered">
                            < ... />
                        </table>

Horizontal Striped

To add stripes to table, add .is-striped class to the table. Note this uses the CSS psudeo selector nth-child. There is not fall back for browsers that don't support this feature.

# Name Age Gender
1 Warren Buffet 90 Male
2 Steve Jobs 50 Male
2 Narine Smith 30 Female 
                       <table class="table is-striped">
                            < ... />
                        </table>

Table Hovered

To add hover effect on each table row, add .is-hovered class to the table.

# Name Age Gender
1 Warren Buffet 90 Male
2 Steve Jobs 50 Male
2 Narine Smith 30 Female 
                       <table class="table is-hovered">
                            < ... />
                        </table>

Table Condensed

To lessen the padding around each table cell, add .is-condensed class to the table.

# Name Age Gender
1 Warren Buffet 90 Male
2 Steve Jobs 50 Male
2 Narine Smith 30 Female 
                        <table class="table is-condensed">
                            < ... />
                        </table>

Table Cell States

To add state indicators to the table elements, add state class to tr of td element. states available .success, .warning and .error.

# Name Age Gender
1 Warren Buffet 90 Male
2 Steve Jobs 50 Male
2 Narine Smith 30 Female 
                        <!-- Add classes to table rows -->
                        
                        <tr class="success">
                            < ... />
                        </tr> 
                        <tr class="error">
                            < ... />
                        </tr> 
                        <tr class="warning">
                            < ... />
                        </tr>
                         
                        <!-- Add classes to table cells -->
                        
                        <td class="success">
                            < ... />
                        </td>
                        <td class="error">
                            < ... />
                        </td>
                        <td class="warning">
                            < ... />
                        </td>
