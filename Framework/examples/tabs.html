Options

Options Defaults Description
debug true Outputs error meaages or warnings to the console
tabs '.tab-btns' Get tab buttons
tabPanes '.tab-panes' Get tab panes that holds the content
animate scaleIn Type animation that should be applied. These are pre written css classes

Methods

Methods Description
Destroy Removes all changes to the page made by initialization
reInitilize Refreshes cached values for a component after Destroy is called
changeTab Opens tab content at index
