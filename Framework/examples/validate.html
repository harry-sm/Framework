Form validation

Form validation requires passing in a validation object with the rules required to validate your form.

Validation Rules

Validation rules are a set of conditions required to validate a field

Name Usage Description
required type : 'required' Checks whether a field is empty
email type : 'email' Checks whether a field is a valid email address
url type : 'url' Checks whether a field is a url
minLength type : 'minLength[val]' Checks whether a field is below a length
Length type : 'maxLength[val]' Checks whether a field is the exact length
maxLength type : 'maxLength[val]' Checks whether a field is longer than a length
not type : 'not[val]' Checks whether a field is not a value
has type : 'has[val]' Checks whether a field has a value
contains type : 'conatins[numbers: val, chars: val, symbols: val]' Checks whether a field contains number of character elements specified
is type : 'is[val]' Checks whether a field matches a value
match type : 'match[field]' Checks whether a field matches another field
checked type : 'checked' Checks whether a checkbox field is checked
selected type : 'selected' Checks whether a radio field is checked
atleast type : 'atleast[number of fields, field]' Checks whether the number of fields specified is not empty
blacklist type : 'backlist[val, val, val, ...]' Checks whether field value is in blacklist
range type : 'range[min, max]' Checks whether value falls withing numeric range
numeric type : 'numeric' Checks whether field value is numeric
alphaNumeric type : 'alphaNumeric' Checks whether field value is alpha-numeric

Add Rule

To add a rule the insance data must be retrive. Then call the addRule() function on the instance. The function takes two parameters ( name of rule, test function). The test function must return true of false. Test function also takes two parameters ( field value, condition value). The condition value is only applicable is bracket syntax is used eg. rule[condition].

                        var instance = $("form").data("form");
                        
                        instace.addRule("ruleName", function (value, condition){
                            return (value == condition);
                        });
                        
                        var validation = {
                        	'field' : {
                            	rules :[
                                    {
                                        type : "ruleName[val]",
                                        prompt : "some message"
                                    },
                                 ]
                            }
                        }

Callbacks

Callbacks specify a function to occur after a specific behavior.

Setting Context Description
valid field Callback on each valid field
invalid field Callback on each invalid field
success form Callback if a form is all valid
failure form Callback if any form field is invalid

Options

Validator settings to modify the validator's behavior.

Options Default Description
debug true Outputs error meaages or warnings to the console
Classes Class Name Class names used to attach style to state
error 'error' Error class attach to form when state is error
success 'success' Success class attach to form when state is success
fieldErr 'field-error' Error class attach to field when state is error
Selectors Selector Name Selectors used to match functionality to DOM
message '.prompt-block' Select element to add error messagaes to
field '.field' Selects field element
group '.field-block' Selects field element container
submit '.submit-btn' Selects submit button
